# 🚀 Task Management System

A full-stack **Task Management System** built using **React, NestJS, and PostgreSQL**.

---

## 🌐 Live Links

- 🖥️ Frontend: https://task-mage.netlify.app/
- ⚙️ Backend API: https://tms-api-7nc6.onrender.com
- 📚 API Documentation: https://tms-api-7nc6.onrender.com/api-docs

---

## ✨ Features

### 🔐 Authentication
- User Signup and Login
- JWT-based authentication
- Protected routes
- Persistent login using Redux + LocalStorage

---

### 📋 Task Management
- Create tasks
- Update tasks
- Delete tasks
- Mark tasks as completed
- Task status tracking:
  - Pending
  - In Progress
  - Completed

---

### 📊 Dashboard
- Board view (Kanban style)
- Table view
- Search tasks
- Filter by status
- Pagination support

---

### 👤 User System
- User profile stored in JWT
- Sidebar shows logged-in user information
- Logout functionality

---

## 🏗️ Tech Stack

### Frontend
- React.js
- TypeScript
- Redux Toolkit
- React Hook Form
- Zod Validation
- Tailwind CSS
- React Router DOM

### Backend
- NestJS
- PostgreSQL
- JWT Authentication
- REST API Architecture
